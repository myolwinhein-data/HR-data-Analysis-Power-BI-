# HR-Data-Analysis (Power BI)

## Project Objective
I studied HR data analysis from free webiner.The project is to analysis headcounts and turnover rate according to the job level,location,gender,department,city and educated level.

## Data Used
- <a href="https://github.com/myolwinhein-data/HR-Data-Analysis-Power-BI-/blob/main/people_data.csv"> people data table</a>
- <a href="https://github.com/myolwinhein-data/HR-Data-Analysis-Power-BI-/blob/main/people_employment_history.csv"> people employment table</a>

## Questions (KPIs)
- Which department have the most Employees?
- The number of headcounts by job Level,Location,cities,gender and age.
- The Employee's education level and marital status.
- Who are leaving the most and why they are leaving?

## Process
- Apply the data transforming and data cleaning and correct data formats.
- Created calender table using calender function.
- Calculated tunover percent,headcounts,starting headcounts and department employee using dax function.
- Designed clear and beautiful dashboard using suitable charts.
- Finally provide meaning insights from the dashboard.
